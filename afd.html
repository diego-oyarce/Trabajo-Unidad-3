Autómata Finito Determinista

Para el correcto uso de la aplicación considerar las siguientes reglas:

  • Antes de ingresar el AFD se debe ingresar el alfabeto de este autómata, de lo contrario no se podrán ingresar elementos al AFD.
  • El primer estado que se ingresa se considerará como estado inicial.
  • Al momento de ingresar un estado se debe indicar si este será final o no.
  • Para obtener la expresión regular del AFD oprima el boton Obtener ER.
